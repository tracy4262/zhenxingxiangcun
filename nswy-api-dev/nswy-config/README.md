
## 简介
eglsc-helper 是java 工具辅助类

## 特征&提供
- BeanByRefMapper bean 之间的拷贝
- 集合操作类 Lists，Maps，Sets，Sorts
- 。。。。。。

 
## 快速开始
 maven 直接引入
           <dependency>
              <groupId>com.github.xphsc</groupId>
              <artifactId>eglsc-helper</artifactId>
              <version>1.2.3</version>
          </dependency>
